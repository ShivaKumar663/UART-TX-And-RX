UART Transmitter & Receiver 

Project Overview
This project implements a Universal Asynchronous Receiver and Transmitter (UART) protocol in Verilog HDL, designed and simulated using  Vivado.  
The design includes:
Baud Rate Generator
UART Transmitter (Tx)
UART Receiver (Rx)

The UART is one of the most widely used serial communication protocols. This project demonstrates the working of asynchronous serial data transmitting and receiving.
Features
Baud Rate Generator for timing control
UART Transmitter 
UART Receiver
Testbench for end-to-end simulation
